# Salesforce Service Management Admin Project
---

## Project Overview
This project implements a Service Management system using Salesforce Admin features.
It helps organizations track service requests, assign technicians, calculate costs,
and analyze performance using reports and dashboards.

## Features Implemented
- Custom Objects (Service Request, Technician Assignment, Payment)
- Validation Rules
- Roll-Up Summary Fields
- Record-Triggered, Screen, and Scheduled Flows
- Reports and Dashboards
- Role-Based Security and Permission Sets

## Tools Used
- Salesforce Developer Edition
- Salesforce Flow Builder
- Reports & Dashboards

## Use Case
Automates after-sales service operations including request tracking, technician work,
payment management, and service analytics.
